## ⚠️Дисклеймер
Проект выкладывается в нерабочей версии, поскольку Terms of Service (ToS) сайта, с которым он работает, не существует. Данный репозиторий представляет собой демонстрацию кода и архитектурных решений.
## Описание
GorzdravBooking - это сервис для автоматизации записи к врачам. Проект предназначен для автоматизации процесса бронирования приемов в медицинских учреждениях.
## Текущее состояние
На данный момент реализовано консольное приложение с интерактивным меню для управления процессом записи. В будущем планируется разработка веб-интерфейса и Telegram-бота для более удобного взаимодействия с системой.
## Запуск проекта
Для запуска проекта необходимо использовать консольное приложение:
```bash
cd src/presentation/
dotnet run
```
