# Консольное приложение

### 1. Создание профиля пациента
- Выберите район и поликлинику, к которой прикреплен пациент.
- Введите необходимые данные. Система автоматически проверит существование профиля в выбранном учреждении.
- При успешной проверке профиль будет создан в системе.

### 2. Настройка временных предпочтений
- Создайте «Пресет» — шаблон удобных для вас дней и времени.
- Для каждого дня недели вы можете указать конкретные промежутки времени.
- Также можно выбрать опцию «Любое время» — в этом случае система запишет вас на первый доступный талон.

### 3. Создание запроса на запись
- В меню «Запись» выберите:
    - Пациента из созданных профилей.
    - Специальность врача.
    - Конкретного врача.
    - Подходящий временной пресет.
  
- После подтверждения система активирует ваш запрос и начнет автоматический поиск подходящих талонов.

### Панель состояния
Всегда следите за верхней строкой меню — это ваша главная панель управления. Она отображает ключевую информацию в реальном времени:

`Title | количество активных запросов | статус работы сервиса | когда будет следующий запрос | 5 последних попыток`
